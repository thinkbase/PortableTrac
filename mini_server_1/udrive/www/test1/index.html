Welcome to Mini Server 1 based on Uniform Server 3.5-Apollo

The server has the following specification:

  • Server shall be portable
  • Only static HTML pages shall be served.
  • The server shall log all web requests.
  • All unused modules shall be disabled.

Disk size around 1MB for a complete Apache 2.2.9 server meeting our specification.

This small mini server is part of a series exploring the flexibility of Uniform Server’s architecture. Primarily intended for educational use and testing they are mini production servers you can place on-line.

Make sure you are running a firewall and check the security settings meet your requirements.

Each mini server has a demo web site or sites providing an example of the server’s capabilities. These are not part of the server’s functionality and can be safely removed. (To clean the server; remove everything in folder www and add your own site)
