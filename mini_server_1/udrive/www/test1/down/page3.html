Running multi-servers

To run multi Apache servers:

1) Create a new folder and copy the contents from server 1 into it.

2) Each server must have a unique port. Change port see page 1.

Note: You can run the mini servers alongside Uniform Server 3.5-Apollo make sure to start Uniform Server first.
