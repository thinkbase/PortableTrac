Welcome to the MiniServer

Test sites included are skeletons providing working examples these demonstrate the mini's capabilities. To change server parameters check out the read me file. Additional support can be found on the Wiki.

 

The server has the following specification

  • Server shall be portable
  • Only static HTML pages shall be served.
  • The server shall log all web requests.
  • All unused modules shall be disabled.

The Uniform Server is a WAMP package that allows you to run a server on any MS Windows OS based computer. It is small and mobile to download or move around and can also be used or setup as a production/live server. Developers also use The Uniform Server to test their applications made with either PHP, MySQL, Perl, or the Apache HTTPd Server.
The MiniServer Line by Uniform Server are smaller cut down versions of the Uniform Server to make a more portable and mobile server environment on the GO. These servers are fully secure and they use Uniform Server's architecture for portability.
